# 🎓 Subject Learning Hub

A colorful and interactive multi-page educational website built using **HTML** and **CSS**. The website provides information about various technology subjects through attractive cards, vibrant gradients, emojis, and easy navigation.

## 🚀 Features

* 🎨 Modern and colorful user interface
* ✨ Glassmorphism card design
* 🌈 Animated gradient backgrounds
* 📱 Responsive layout
* 🔗 Hyperlinked navigation between pages
* 🏠 Back-to-home button on every subject page
* 😊 Emoji-enhanced user experience
